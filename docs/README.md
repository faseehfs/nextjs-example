# Docs

## Commit message prefixes

| Prefix | Use Case | Example |
| :--- | :--- | :--- |
| **feat:** | A new feature | `feat: add dark mode toggle` |
| **fix:** | A bug fix | `fix: resolve tsconfig alias error` |
| **docs:** | Documentation only | `docs: update readme installation steps` |
| **style:** | Formatting/UI (no code logic change) | `style: fix button alignment` |
| **refactor:** | Code change that neither fixes a bug nor adds a feature | `refactor: simplify auth logic` |
